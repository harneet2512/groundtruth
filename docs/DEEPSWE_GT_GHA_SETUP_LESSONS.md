# DeepSWE GHA Setup Lessons — inefficiencies SWE-Live-Lite hit, so DeepSWE avoids them day one

> Each was found + fixed on `swebench_300task.yml` this session. DeepSWE must inherit the FIX, not the
> anti-pattern. Branch `gt-trial`.

| # | Inefficiency / anti-pattern | Where in SWE-Live-Lite | Why it was bad | Final pattern | DeepSWE prevention (risk · guard · failure class · test/lint) |
|---|---|---|---|---|---|
| 1 | per-task ~130MB e5 model download/copy | early host-mode setup-eval | per-task network + flake + slow | **bake the model in the substrate image** (`Dockerfile.gt-substrate:60`) | DeepSWE risk: the OLD `deepswe_full` fetched e5 per-run from release assets · guard: model baked, `GT_REQUIRE_EMBEDDER=1`, no runtime download · class `EMBEDDER_MODEL_ROOT_DIVERGENCE` · test: substrate self-test (`Dockerfile:90-92`) |
| 2 | per-task pip install / dep provisioning | host setup-eval | per-task install + version drift | **bake deps in substrate; `--validate` fails-closed if not baked** | risk: OLD `deepswe_full` `pip install datacurve-pier`+LSP per run · guard: `validate_proof_env` rejects un-baked · class `SUBSTRATE_NOT_PORTABLE` · test: `test_portable_substrate.py` |
| 3 | host/image split (graph on host, agent in container) | host gt-index + host LSP, agent in task image | the conan sem=0 confound; graph the agent reads ≠ the proven graph | **run the whole pipeline IN the substrate via `gt-run-proof`; agent consumes `/gt_artifacts`** | risk: OLD `deepswe_full` host gt-index/LSP + agent in container · guard: `assert_container_boundary`, host run_v74 fails in proof · class `FINAL_PIPELINE_HOST_SPLIT_FAIL` · test: `test_container_lockdown.py` |
| 4 | redundant GT paths | v1r/run_v74 (OH) vs gt_intel/gt_hook (mini-swe) — two engines | divergence; fixes land in one; ast Python-only | **ONE engine — unify mini-swe onto v1r/graph-SQL; retire gt_hook ast** (done this session in `gt_mini_patch`) | risk: gt_intel/gt_hook still importable · guard: per-turn evidence is graph.db SQL, cross-lang · class — · test: `test_deepswe_xlang_evidence.py` (RED→GREEN Go) |
| 5 | mutable image tags as proof input | `:latest`-style | non-reproducible proof | **pin `@sha256:` immutable digest** | risk: DeepSWE must pin `GT_SUBSTRATE_DIGEST` · guard: `@sha256:` assertion, `require_pinned_substrate=1` · class `GT_SUBSTRATE_DIGEST_MISSING` · lint: `test_portable_substrate.py::test_workflow_substrate_pinned_by_digest` |
| 6 | artifact upload only on success | upload step without `if: always()` | failed-run artifacts lost → can't diagnose | **`if: always()` upload + emit contracts even on failure** | risk: DeepSWE upload must be `if:always` · guard: upload step condition · class — · lint: `workflow_lint.py` |
| 7 | image pull fragility | docker/login + per-task pull, no retry → ~31% login + 3/300 pull fails | flaky infra fails look like GT fails | **6-try login backoff + 8-try exp-backoff pull, GHCR-first, classified** | risk: 113 cold ECR pulls rate-limit at max-parallel 20 · guard: GHCR-first + retry · class `GT_SUBSTRATE_PULL_FAIL` / `TASK_IMAGE_PULL_FAIL` · verify the 113 GHCR cache exists |
| 8 | gates_only / live path divergence | risk of two prefixes | gates prove a different path than live | **same runtime/graph/brief prefix; gates_only just skips the agent** | risk: DeepSWE has no dry mode · guard: add a proof-only/dry mode sharing the substrate step · class — · lint: workflow structure test |
| 9 | host-primary brief/scoring in proof | OH wrapper host-primary `run_v74` → 10/10 agent crash | host GT scoring in proof = forbidden + crashed the agent | **agent CONSUMES the in-container `/gt_artifacts/brief.txt`; no host scoring in proof** | risk: gt_agent must consume `brief.txt`, not host-score · guard: `assert_container_boundary` in run_v74 · class `FINAL_PIPELINE_HOST_SPLIT_FAIL` · test: brief-consume path |
| 10 | LSP cert count vs final-graph stamp mismatch | gate read the LSP_METRICS stdout line, not the graph → `lsp_resolved=0` universal | green-looking cert while the metric read 0 | **gate counts `resolution_method='lsp'` from the FINAL graph + cross-checks the cert** | risk: DeepSWE per-lang; non-Python = no LSP edges (expected, classified) · guard: `lsp_stamp_check` · class `LSP_STAMP_DROPPED` / `LSP_GATE_MEASUREMENT_MISMATCH` · test: `test_lsp_stamping.py` |
